# Type: ```ConnInfo```  
## Typical construct in response (payload)  
```
{  
    "token": Token<String>,
    "direction": Direction<String>
    "port": Port<int>
}  
```
### Fields:  
  
```token``` - authorization token for starting the transfer  
```direction``` - either ``GET`` or ``POST``
```port``` - port of the server
